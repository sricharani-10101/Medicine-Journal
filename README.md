# Medicine Journal - Health Assistant

# Objective
A small place is designed to find some health assistance for common health problems that occur during any point in life. This program consists of a few common symptoms that can be cured by simple medication and some tips to avoid in order to stay healthy. If the condition is severe or the symptoms do not match, the program advises visiting a doctor for further medical assistance.

# Features of this Project
1. Symptom Checker: Accepts user input for a wide variety of common illnesses that are mentioned on the database.
2. Medicine Suggestions: Provides specific over-the-counter and easily available common medicine names for relief.
3. Diet Advice: distinct "Avoid" lists for each symptom to help the user recover faster (e.g., avoiding caffeine or spicy foods).
4. Safety Fallback: Identifies when a symptom isn't in the database and advises professional medical help.
5. User-Friendly 
6. Interface: Simple text-based interaction suitable for beginners.

# Tools Used
1. Language: Python
2. Editor: Python IDLE.
3. References: Modules and medicine data from online sites

# Steps to Install and Run
1. Install python
2. Open a new file in IDLE Shell
3. Create a filename MedicineJournal.py
4. Write code and save it
5. Execute code
6. Enter a symptom and record the output

# Instruction for Testing
1. Start the program 
2. Enter the symptom
3. If there is data available it gives medication
4. If data is unknown suggests to visit a doctor
5. The program automatically ends after giving medication
6. For another symptom run it again
